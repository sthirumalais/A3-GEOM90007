# Bird Watch Visualization App

## Data Source & Acknowledgement

This project uses bird sighting data obtained from **BirdLife Australia’s Birdata** database, accessed via the **Atlas of Living Australia (ALA)**.  
The dataset includes over **32,000 bird observation records** from the **City of Melbourne (LGA: Melbourne)** area.

We gratefully acknowledge **BirdLife Australia** and the **Atlas of Living Australia** for making these data openly available.  
This dataset has been used solely for educational purposes as part of the Bird Watch Visualization App project.
